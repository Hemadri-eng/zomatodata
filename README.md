# zomatodata
This project performs Exploratory Data Analysis (EDA) on Zomato restaurant data using Python libraries like Pandas, Matplotlib, and Seaborn. The goal of this project is to analyze restaurant trends, customer preferences, ratings, online ordering patterns, and spending behavior.
This project analyzes Zomato restaurant data to understand customer behavior, restaurant performance, and ordering trends. After cleaning and preprocessing the dataset, exploratory data analysis was performed to identify popular restaurant types, voting and rating patterns, average customer spending, and the impact of online versus offline ordering. Insights were presented using visualizations such as count plots, line charts, histograms, box plots, and heatmaps to support data-driven conclusions.
